External Node Server

This node-powered proxy provides another route to browse the open web. Paste the address you want to visit to let the server fetch it for you.

Features

  • Play YouTube videos by prefixing with https://invidio.us/.
  • Access blocked IO games.
  • Load script-heavy webpages.
  • Use services that rely on WebSocket connections.
